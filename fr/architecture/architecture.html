Comment ça marche ?¶

L’architecture d’Iris est assez simple.

Iris frontend exposes l’application web qui utilise le service d’authentification pour permettre à l’utilisateur de se connecter. Il utilise aussi Iris API pour gérer les idées et autres ressources.

Iris API exposes toutes les méthodes pour gérer les données utilisées par l’application web. Les endpoints de l’api sont protégées par le même système d’authentification que le frontend, car attend le jeton de l’utilisateur. Il utilise aussi la base de données et le service de stockage d’images pour stocker données, images et informations.
