Contribute
==========

You have many ways to contribute in this project, you can :

- Open issues to :
    - report an issue
    - request an improvement
    - ask some questions
- Discuss on these opened issues
- Contribute in the projects or the documentation
    - `Iris API <https://github.com/RoadTripMoustache/iris-api/blob/main/CONTRIBUTE.md>`__
    - `Iris Docs <https://github.com/RoadTripMoustache/iris-docs/blob/main/CONTRIBUTE.md>`__
    - `Iris Frontend <https://github.com/RoadTripMoustache/iris-frontend/blob/main/CONTRIBUTE.md>`__

Every action you can take will help Iris to live and grow !
